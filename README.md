# 아이 이름 후보 정하기

1. 대법원 인명용 한자표에 해당하는 한글로 이름짓기
2. 받침 없는 한글은 외국인도 따라하기 쉬움


## 1차 이름 후보
```
python baby_name.py > name_001.md
```

## 2차 이름 후보
`name_001.md`파일에서 괜찮은 이름은 `**가나다**`로 선택함

```
cat name_001.md | tr " " "\n" |\
    grep -E '\*\*.*\*\*' |\
    sed -e 's/\*//g' > name_002.md
```

## 3차 이름 후보
괜찮은 이름을 한번 더 선택함
